เริ่มต้นใช้งาน

SadTalker Studio คือเครื่องมือสร้างวิดีโอ Talking Head — ใส่รูปหน้าคน + ข้อความ แล้วระบบจะสร้างวิดีโอที่หน้าขยับปากพูดตามข้อความให้อัตโนมัติ

3 ขั้นตอนง่ายๆ:

  1. เลือกรูปภาพ — อัปโหลดรูปหน้าคน หรือเลือกจากตัวอย่าง (ต้องเห็นหน้าชัด 1 หน้า)
  2. พิมพ์ข้อความ — ที่ช่องด้านล่าง แล้วเลือกเสียงพูด (ไทย/อังกฤษ/ญี่ปุ่น/เกาหลี/จีน)
  3. กด Generate — รอประมาณ 5-10 นาที จะได้วิดีโอ MP4 พร้อมดาวน์โหลด
Preset คืออะไร?

Preset คือชุดค่าตั้งสำเร็จรูปที่กำหนดคุณภาพและสไตล์วิดีโอ:

เร็ว (Draft) เร็วสุด ตัดเฉพาะหน้า ไม่ enhance — เหมาะทดสอบ
มาตรฐาน เต็มภาพ + GFPGAN enhance — สมดุลคุณภาพ/เวลา
คุณภาพสูง เต็มภาพ + GFPGAN + หัวขยับ — คุณภาพสูงสุด
การตลาด เต็มภาพ + GFPGAN + expression เข้ม — เหมาะโปรโมท

สร้าง Preset เองได้ที่เมนู "จัดการ Preset"

ตั้งค่าขั้นสูง
Expression Scale ปรับความเข้มของการแสดงออก (0.5=นิ่ง, 1.0=ปกติ, 2.0=เวอร์มาก)
Pose Style 46 รูปแบบการขยับหัว (0-45) แต่ละเลขให้ลักษณะต่างกัน
Still Mode ON = หัวนิ่ง ขยับแค่ปาก/ตา (แนะนำ) | OFF = หัวขยับตามเสียง
Preprocess crop=เฉพาะหน้า, full=เต็มรูป, resize=ย่อทั้งรูป
Face Enhancer GFPGAN หรือ RestoreFormer — ทำให้หน้าคมชัดขึ้น
BG Enhancer RealESRGAN — ขยายความละเอียดพื้นหลัง 2 เท่า
Yaw / Pitch / Roll กำหนดมุมหมุนหัวเอง (ซ้าย-ขวา / ก้ม-เงย / เอียง)

ปรับ ความเร็ว (ช้ามาก — เร็วมาก) และ ระดับเสียง (ต่ำ — สูง) ได้

หรือ อัปโหลดไฟล์เสียง (.mp3 / .wav) ของตัวเองแทน TTS ก็ได้

ใช้งานผ่าน Command Line

นอกจากหน้าเว็บ ยังใช้คำสั่งในเทอร์มินัลได้:

# เสียงผู้หญิงไทย (default)
./ss "สวัสดีค่ะ" รูปหน้า.jpg output.mp4
# เสียงผู้ชายไทย
VOICE=th-TH-NiwatNeural ./ss "สวัสดีครับ" face.jpg
# เสียงอังกฤษ
VOICE=en-US-JennyNeural ./ss "Hello!" face.jpg

การจัดการข้อมูลส่วนบุคคล (PDPA)

SadTalker Studio เก็บข้อมูลเฉพาะที่จำเป็นสำหรับการให้บริการ ตาม พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล (PDPA) และ GDPR

ข้อมูลที่เก็บ ชื่อผู้ใช้, รูปที่อัปโหลด, วิดีโอที่สร้าง, ข้อความ TTS, ประวัติการใช้งาน
วัตถุประสงค์ สร้างวิดีโอ Talking Head ตามคำขอของผู้ใช้เท่านั้น
การเก็บรักษา ข้อมูลเก็บในเซิร์ฟเวอร์ ไม่ส่งต่อบุคคลที่สาม
ระยะเวลา จนกว่าผู้ใช้จะลบ หรือขอลบบัญชี
